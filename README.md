# qs-netcat

```
qs-netcat [options]
    -s, --secret <string>        Qsocket unique id
    -e, --exec   <command>       Executes the given command
    -p, --port   <int>           Port to expose/forward QSRN
    -T, --tor                    Use TOR.
    -i, --interactive            Interactive login shell (TTY) [Ctrl-e q to terminate]
    -g, --generate               Generate a Secret (random)
    --no-tls                     Disable TLS encryption.
    -v, --verbose                Verbose output.
```
